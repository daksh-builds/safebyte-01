# 🔐 Cyber Security Toolkit

A modern cybersecurity toolkit built with Next.js, focused on providing essential security utilities in one place. This project combines multiple security tools to help detect vulnerabilities, monitor system activity, and enhance overall protection.

---

## 🚀 Features

* 🔍 **Vulnerability Assessment Tool**
  Analyze systems and identify potential security weaknesses.

* 📂 **File Crawler**
  Efficiently scan and navigate files for analysis and monitoring.

* 🔑 **Smart Password Manager**
  Securely store and manage credentials with intelligent handling.

* 🦠 **Malware Detection Tool**
  Detect suspicious or malicious files and activities.

* 🖥️ **Screen Monitor**
  Track real-time screen activity for security purposes.

* 🔌 **USB Detector**
  Monitor and detect USB device connections to prevent unauthorized access.

---

## 🛠️ Tech Stack

* **Frontend:** Next.js
* **Runtime:** Node.js
* **Package Managers Supported:** npm, yarn, pnpm, bun

---

## ⚡ Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open your browser and go to:

```
http://localhost:3000
```

Start editing the app by modifying:

```
app/page.js
```

The page will auto-update as you make changes.
